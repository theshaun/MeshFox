# MeshFox

MeshFox is an Ubuntu based distribution for Luckfox boards with LoRa radios based on the Foxbuntu work by Femtofox.

MeshFox focuses on expanding functionality to Luckfox based boards such as the [Femtofox](https://github.com/femtofox/femtofox) for use with pyMC_Repeater.
